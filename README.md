# YT-Creator-Kit

### One-liner
An automated toolkit designed to streamline content creation and metadata optimization for the **Brainrot Factory** YouTube channel.

### Tech Stack
* **Language:** Python 3.10+
* **APIs:** YouTube Data API v3
* **Tools:** Git, VS Code, CapCut

### Deployment Instruction
1. Clone the repository: `git clone https://github.com/andriidovhanychpp2023-crypto/yt-creator-kit.git`
2. Create and activate virtual environment: `python -m venv venv` потім `venv\Scripts\activate`
3. Install dependencies: `pip install -r requirements.txt`

### MVP Goals
* **Script Generator:** Automated creation of viral hooks and quiz-style scripts.
* **Auto-Metadata:** Generating optimized titles and tags for the US market.
* **Trend Tracker:** Monitoring Brainrot/Aura meme trends.
